### 高低检测

| 项目 | 标准 | 精度要求 |
| :--: | :--: | :------: |
| 高低 | $0$  | $\pm1.5$ |

- 输入：倾角仪
- 方法：加速度场，速度场，位移场
- 相关教程：http://www.starlino.com/imu_guide.html
- 输出：找出铁轨当前倾角，计算高低误差，图形化显示
